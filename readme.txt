== Description ==

From wordpress we need to send many transactional emails like 

* User registration.
* Forget password 
* User registeration notification and yes when we use advance plugins it send all notification by email to admin
* In woocommerce we need to send email to buyers customers for their orders 
* And order status changes
	and we many more cases.


Sakshamapp ES the complete Email System for startups and enterprises and using this you can manage smootly the delivery of all these transnational emails. 

The integration is very easy you only need to follow few steps In case it do not work please talk me on skype gcsplindia I will do it free of cast otherwise just post a message I will explain.

You need to register on the sakshamapp es system http://app.sakshamapp.com/ and get public key and private key and input the key in the setting page.

Details about the sakshamapp es is available at https://www.sakshamappinternational.com/sakshamapp-es/

== Frequently Asked Questions ==

= How it send email? =

It uses rest api and send email via Sakshamapp ES

= Where email log is stored =

In the wordpress installation all log is stored in your wordpress database you can check from the system backend /wp-admin/admin.php?page=sakshamapp_email_log

= Does sakshamapp es store emails log in their server =

Sakshamapp ES deliver email immidiatly it dont store any thing. However we maintain the email deliver log for 3 days for any kind of bug recovery and in this we keep only email header.
